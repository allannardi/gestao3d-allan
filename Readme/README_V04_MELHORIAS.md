# Gestão 3D Allan v04 - Melhorias de uso

Incluído nesta versão:

1. Categorias de peças flexíveis, com opção de criar nova categoria durante cadastro/edição da peça.
2. Filamentos/cores no pedido, com botão para adicionar mais filamentos ao pedido.
3. Aba Peças agora mostra pedidos relacionados à peça, quantidade vendida, faturamento e lucro estimado.
4. Cards de pedidos redesenhados com número maior, peça em destaque, quantidade e status visual com bolinha colorida.
5. Duplicar pedido agora abre modal perguntando se é o mesmo cliente, outro cliente ou novo cliente.
6. Dashboard com pedidos fechados no mês, gráfico de barras para peças mais vendidas, status visual e lista de pedidos abertos.

Observação:
O cálculo de custo da peça continua baseado no cadastro principal da peça. Os filamentos adicionais no pedido são tratados como informação operacional de cor/material usado naquele pedido.
