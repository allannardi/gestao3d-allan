# V11.08 — Dashboard mobile corrigida

## Correção principal
A página Início no modo mobile agora segue a mesma organização aprovada no desktop.

## Nova ordem no mobile
- Resumo do mês / cards principais
- Pedidos por status
- Pedidos abertos por peça
- Vendas por mês
- Peças com maior faturamento
- Clientes com maior faturamento

## Gráfico no mobile
- O gráfico Vendas por mês agora também aparece no mobile.
- Foram mantidas vendas e lucro em barras, com margem de lucro em linha.

## Ranking no mobile
- Peças deixou de ser ranking por quantidade e passou a ser por faturamento.
- Clientes com maior faturamento foi adicionado no mesmo padrão visual mobile.
