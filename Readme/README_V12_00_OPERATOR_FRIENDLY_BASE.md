# V12.00 — Operator Friendly Base

## Objetivo
Primeira fase para deixar o Gestão 3D mais fácil e intuitivo para operadores com qualquer nível técnico.

## Alterações
- Sidebar atualizada para versão 0.12.
- Adicionada seção "Como usar esta tela?" em:
  - Pedidos
  - Peças
  - Filamentos
  - Acessórios
  - Clientes
  - Configurações
- Adicionada legenda dos status na tela de Pedidos.
- Adicionadas dicas/captions nas áreas principais de cadastro.
- Adicionados textos de ajuda nos campos técnicos principais:
  - quantidade vendida
  - valor unitário
  - desconto
  - frete
  - filamento/rolo usado no pedido
  - quantidade produzida no lote
  - tempo de impressão do lote
  - tempo de pós-processamento
  - custo de embalagem
  - peso original do rolo
  - valor da compra do filamento
  - energia/hora
  - depreciação/hora
  - margem padrão
  - meta lucro/hora
  - pós-processamento/hora

## Segurança
- Não altera banco de dados.
- Não cria nem remove tabelas.
- Não altera lógica de cálculo.
- Não altera dados cadastrados.
- Foco apenas em orientação, clareza e usabilidade.
