# V14.01 — Ajustes de Admin

## Objetivo
Organizar a tela de Configurações para separar os cadastros normais de impressoras das ações pontuais de manutenção.

## Alterações
- Criada a nova seção `Ajustes de Admin` em Configurações.
- Movido `Ajuste em lote` para a nova seção.
- Movido `Ajuste pontual de datas dos pedidos antigos` para a nova seção.
- A seção `Impressoras` fica focada em impressoras cadastradas e cadastro de nova impressora.
- Sidebar renumerada para `0.14.01`.

## O que não mudou
- Nenhuma regra de negócio foi alterada.
- Nenhum dado é alterado automaticamente.
- Nenhum cálculo foi alterado.
- Os ajustes continuam funcionando com confirmação antes de executar.

## Observação
Por enquanto a seção aparece normalmente. Em uma fase futura, ela deve ficar visível apenas para usuários com perfil Admin.
