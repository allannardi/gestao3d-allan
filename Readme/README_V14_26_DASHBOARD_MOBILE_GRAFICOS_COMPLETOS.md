# V14.26 — Dashboard mobile com gráficos completos

## Objetivo
Corrigir a diferença entre a Dashboard desktop e mobile após a consolidação da v14.24/v14.25.

## Alteração principal
A versão mobile da Dashboard agora também exibe:
- Faturamento por impressora;
- Distribuição do faturamento;
- Utilização das impressoras por mês.

## O que não mudou
- Banco de dados.
- Cálculos.
- Filtros.
- Regras de negócio.
- Services.
- Visual desktop.

## Validação recomendada
- Abrir a Dashboard no desktop.
- Simular tela mobile pelo navegador ou abrir no celular.
- Conferir se aparecem:
  - Vendas por mês;
  - Faturamento por impressora;
  - Distribuição do faturamento;
  - Utilização das impressoras por mês;
  - Peças com maior faturamento;
  - Clientes com maior faturamento.
